# repositorio_prompt
Repositorio de Plantillas de Prompts -Este repositorio contiene un conjunto de herramientas en forma de **plantillas de prompts** optimizadas. 
Repositorio de Plantillas de Prompts - Tecnicatura en IA
 ¿De qué se trata este proyecto?
 Su objetivo es ayudar a cualquier estudiante o científico de datos a interactuar con Inteligencias Artificiales (como ChatGPT, Claude o Gemini) para resolver tareas comunes de programación y análisis de datos de forma rápida y sin errores.

## 🗂️ Estructura del Repositorio
El repositorio está dividido en carpetas que siguen el orden lógico de un proyecto de datos:
1. `01_preprocesamiento/`: Contiene herramientas para limpiar los datos y manejar valores nulos o atípicos.
2. `02_modelado_ml/`: Contiene instrucciones para entrenar modelos de Machine Learning.

## 🛠️ Cómo usar las plantillas
1. Ingresa a la carpeta que necesites según tu tarea.
2. Abre el archivo de la plantilla y **copia el texto del prompt**.
3. Reemplaza las palabras que están entre corchetes `[ ]` con los datos reales de tu proyecto.
4. Pega el resultado en tu IA de preferencia.
